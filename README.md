MathFlow es un lenguaje de programación interpretado desarrollado en **Racket** utilizando **EOPL (Essentials of Programming Languages)** y **SLLGEN** para el análisis léxico y sintáctico.

El proyecto implementa un intérprete capaz de ejecutar programas escritos en MathFlow, soportando variables, constantes, funciones, estructuras de control, listas, diccionarios y álgebra simbólica.
